## Get started

1. [Run gateway](vinhlq/mht-zigbee-gateway-js/blob/master/README.md)

  * > set MQTT_HOST=mht.vn
  * > set MQTT_PORT=1883
  * > sh run_cloud.sh
2. Build
  > npm run build
3. Run
  > npm run server
